## 📋 전체 구축 절차 (10단계)

### 1️⃣ 프로젝트 초기화 및 기본 구조 생성

```bash
cd NEW_BRAND_NAME
npm install framer-motion next-intl lucide-react @next/bundle-analyzer
mkdir -p src/{components,data,utils} public/{images,styles} scripts
```

### 2️⃣ 브랜드 설정 파일 적용 및 커스터마이징
